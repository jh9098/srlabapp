# TODO_MVP_GAPS

이 문서는 이제 **무엇이 아직 기능 미완성인지**와 **무엇이 외부 설정/프로덕션 준비인지**를 명확히 구분하기 위해 사용합니다.

## 1. 현재 판정 요약

### 1-1. 로컬 MVP 완료 여부
**완료로 판단**합니다.

현재 저장소는 아래를 만족합니다.

- 백엔드 핵심 API 실행 가능
- support state engine 테스트 통과
- 관심종목 / 종목 상세 / 홈 / 테마 / 알림 흐름 존재
- 관리자 로그인 + 관리자 운영 API 존재
- 관리자 웹에서 로그인 후 운영 데이터 점검 가능
- 디바이스 토큰 등록 API 존재
- notifications 저장 및 signal_events 연동 존재
- FCM 설정이 있으면 실제 전송 시도 가능
- FCM 설정이 없으면 fallback 동작 존재
- 루트 Flutter 구조 기준으로 문서 정렬 완료
